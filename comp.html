Well, it's official: I'm now a professional geek. OK, so I get paid to write computer programs which may or may not be the same thing :-) Either way, here's the usual laundry list of useful and/or interesting links related to my day-to-day computing.

Linux

  • I am Linux user #180140 at the Linux Counter. Eighteen years of using Linux :-)
  • My (former) UBC workstation runs Fedora 20 on a 6-core Phenom II AMD64 system with 8 GB of RAM.
  • At home I have an older dual-core AMD64 with a mere 4 GB of RAM running Ubuntu as a server and sometime development machine. It's currently on 12.04 LTS, Precise Pangolin - partly because it's an LTS release but mostly because pangolins are cool animals (and the Ubuntu desktop artwork is super cute). System drive is a 64 GB SSD with a 2- and a 3-TB data drive.
  • The Fedora Forum and Ubuntu Forums are useful places for help, info and advice.
  • Linux news at Linux Today and Linux Weekly News.
  • For those who care, I currently use Cinnamon as my desktop environment.

Hardware

  • In late 2012 I took the plunge and bought a Mac, a neat little 13-inch Macbook Air with a 128 GB SSD and 8 GB RAM, which has been a phenomenal machine. I didn't hate OS X as much as I expected :-) and I could almost say I now like it, though I miss some of the GNU versions of utilities. It's up to date with OS X 10.10 Yosemite, which seems fine to me. If I had a complaint it would be the screen - it's not good enough for accurate colour work when processing photos.
  • Which neatly segues into... photo processing and management duties are handled by DxO OpticsPro and ACD See on a Windows 7 machine, reasonably specced with an Intel quad-core CPU running at 3.1 GHz, 8 GB of RAM and a 128 GB SSD as the system drive. The 2 TB data drive is starting to look a little full (the perils of shooting raw files with 2 dSLRs). The monitor is a decent Dell 2209WA, which I trust for colour. The graphics card is a puny Radeon 5450 but it's fanless which I value more than performance.

Astronomical software

  • My former day job: SCUBA-2 data reduction software. Progress and updates can be found on the Pipelines and Archives blog, while pretty pictures are on the official SCUBA-2 blog.
  • The UK Starlink project (now disbanded). Starlink software can be obtained from the JAC Starlink page.
  • The ORAC-DR data reduction pipeline and its post-processing relative PICARD.
  • A conference a year courtesy of ADASS.
  • FITS: the standard (-ish) astronomical data format.

Other software

  • The real way to make web pages look good: the W3 site for CSS, and Eric Meyer's page on CSS.
  • The real way to make your printed page look good: LATEX. Unbeatable for maths typesetting.
  • OK if you must use an office suite, then consider LibreOffice. It comes with its own PDF writer, so there's no excuse for sending .doc files by email (of course, there's rarely an excuse to send .doc files by email...). Plus it supports the Open Document Format.
  • But nothing beats Emacs for doing everything else :-)
  • My must-have apps are (in no particular order):
    • Firefox and/or Google Chrome for the web
    • Thunderbird for email
    • Emacs for editing anything
    • Xfig for graphics
    • TeX/LaTeX for documentation (+dvips/dvipdf)
    • Evince and gv for viewing PDF and PostScript files respectively
    • The Gimp for photos etc
    • GQView and Gwenview for image viewing
    • LibreOffice comes in handy for the odd Word doc, making posters and presentations for conferences
