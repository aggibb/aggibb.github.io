Computers etc

Well, it's official: I'm now a professional geek. OK, so I get paid to write computer programs which may or may not be the same thing :-) Either way, here's the usual laundry list of useful and/or interesting links related to my day-to-day computing.

Linux

  • I am Linux user #180140 at the Linux Counter. I first installed Linux in May 1996, which makes it now twenty years of using Linux :-) My work machine at the time was a Pentium 133 with 32 MB of RAM and a 1 GB HDD. I think it ran Slackware, but I may have switched to Red Hat at some point.
  • My (former) UBC workstation runs Fedora 20; at home I'm running Ubuntu 12.04 LTS, Precise Pangolin though I'll be upgrading to 16.04 (Xenial Xerus) soon.
  • The Fedora Forum and Ubuntu Forums are useful places for help, info and advice.
  • Linux Weekly News remains the best place for news; Phoronix for hardware-related stuff.

Hardware

  • My Linux box is an 8-year old dual-core AMD64 running at 2.2 GHz with a mere 4 GB of RAM running Ubuntu as a server and sometime development machine. The system drive is a 64 GB SSD with a 2- and a 3-TB data drive which are used to back up all of our photos from the Windows machine.
  • In late 2012 I took the plunge and bought a Mac, a neat little 13-inch Macbook Air with a 128 GB SSD and 8 GB RAM, which has been phenomenal - the best laptop I've ever owned by far. I even didn't hate OS X as much as I expected :-) and I could almost say I now like it, though I miss some of the GNU versions of utilities. It's up to date with OS X 10.11 El Capitan, which seems fine to me. My only real complaint is the screen — it's not good enough for accurate colour work when processing photos. Which is why, after three years, I bought a nice little 15" Macbook Pro :-) Oh, that screen... but man does it get hot on my lap.
  • Of course, everyone has to have Windows machine somewhere in their closet. It's reasonably specced with an Intel quad-core CPU running at 3.1 GHz, 8 GB of RAM and a 128 GB SSD as the system drive. The 3 TB data drive is filling up fast thanks to shooting raw files with 3 cameras). The monitor is a decent Dell 2209WA, which I trust for colour. The graphics card is a puny Radeon 5450 but it's fanless which I value more than performance. Currently it's running Windows 7 (which is a decent enough OS), and is scheduled for its free upgrade to Windows 10 sometime "soon". And I'll decide when that will be, thank you Microsoft.

Other software

  • The real way to make web pages look good: the W3 site for CSS, and Eric Meyer's page on CSS.
  • The real way to make your printed page look good: LATEX. Unbeatable for maths typesetting.
  • OK if you must use an office suite, then consider LibreOffice. It comes with its own PDF writer, so there's no excuse for sending .doc files by email (of course, there's rarely an excuse to send .doc files by email...). Plus it supports the Open Document Format.
  • But nothing beats Emacs for doing everything else :-)
  • My must-have apps on Linux are (in no particular order):
    • Firefox and/or Google Chrome for the web
    • Thunderbird for email
    • Emacs for editing anything
    • Xfig for graphics
    • TeX/LaTeX for documentation (+dvips/dvipdf)
    • Evince and gv for viewing PDF and PostScript files respectively
    • The Gimp for photos etc
    • GQView and Gwenview for image viewing
    • LibreOffice comes in handy for the odd Word doc, making posters and presentations for conferences (though to be honest, I find Google Docs is good enough for basic documents)
