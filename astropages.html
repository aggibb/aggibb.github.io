JCMT and related

  • The JCMT is now run by the East Asian Observatory.
  • Download JCMT data from the OMP pages or the archive at CADC.
  • Old-style JCMT data reduction with SPECX and SURF.
  • New-style JCMT data reduction with SMURF for SCUBA-2 and ACSIS

BIMA and related

  • BIMA home page. BIMA is no more: check out the CARMA home page instead.
  • BIMA data is reduced with MIRIAD. Note that BIMA data should be reduced with the older V3 series of MIRIAD. CARMA data reduction should use V4.
  • Other useful stuff includes WIP and various other things on the LMA software page.

Useful utilities

  • Molecular spectroscopy:
    • The JPL Molecular Spectroscopy Home Page.
    • NIST recommended rest frequencies for Observed Interstellar Molecular Microwave Transitions.
    • The Cologne Database for Molecular Spectroscopy.
  • The Skyview Virtual Observatory — see what your favourite part of the sky looks like at various wavelengths.
  • Convert B1950 coordinates to J2000 and more with the NED Coordinate & Galactic Extinction Calculator.
  • Various time-related services from the USNO, including a Sidereal Time calculator.
  • Search astronomical catalogues with VizieR.
